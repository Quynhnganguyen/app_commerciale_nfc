- Description générale de l'application:

Il s'agit d'une application qui permet à un client de scanner des
produits via NFC puis de les comparer, les ajouter à ses produits
favoris ou les ajouter à sa liste noire de produits. Les clients
peuvent aussi créer leur liste de courses à faire. Les magasins, quant
à eux, peuvent alimenter la base de données produits.

Qu'est que NFC?:

C'est une technologie de communication sans-fil à courte portée et
haute fréquence, permettant l'échange d'informations entre des
périphériques jusqu'à une distance d'environ 10 cm.

Comment va-t-on utiliser la techologie NFC?:

On va stocker sur nos cartes NFC des données concernant les
produits. Ces données seront introduites sur les cartes grâce à de
simples fichiers textes.

Langages utilisés:
-Pour l'application générale => Java Android
-Pour la base de données produits => SQL, Php, Html
-Pour la base de données clients => SQLite
